    Disclose GenAI on First Session

    requirement 0 instruments

    A consumer-facing GenAI service must display a clear, prominent disclosure that the consumer is interacting with an AI system at the start of each session.

    What Counts

    • Modal or banner shown on first session that names "AI" or "generative AI" plainly
    • Re-display on session resumption after a defined inactivity threshold (e.g., 30 minutes)
    • Disclosure is in the user's preferred language when supported by the product

    What Does Not Count

    • Burying the notice in a privacy policy or terms-of-service link only
    • Disclosure phrased ambiguously ("smart assistant," "automated helper") without the word "AI"
    • One-time disclosure across multiple unrelated sessions

    Implementing Instruments

    No instruments currently implement this.
