PubLedge schemas

Machine-readable contracts for PubLedge source records and generated representations.

  • instrument.schema.json: polymorphic JSON Schema 2020-12 contract for instrument frontmatter.
  • record.schema.json: contract for each generated record.json.
  • context.jsonld: JSON-LD context binding PubLedge terms to gist.
  • jia.schema.json and rma.schema.json: compatibility schemas retained for existing references.
