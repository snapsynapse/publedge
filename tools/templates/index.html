How to use

  1. Click Download .md to get the raw template.
  2. Replace every {{variable_name}} with the value applicable to your situation.
  3. Update the frontmatter id from PL-TPL-... to a fresh PL-JIA-NNNN or PL-RMA-NNNN permanent identifier.
  4. Set status: draft and add the parties' names under parties:.
  5. Submit a pull request against github.com/snapsynapse/publedge placing the file under data/registry/jia/ or data/registry/rma/.
  6. The CI hash check will fail until you also add the file to MANIFEST.yaml and run ./scripts/validate-hashes.sh --update.
